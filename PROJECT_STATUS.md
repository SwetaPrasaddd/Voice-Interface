### 🎮 How to Use

1. **Setup**: 
   - Install dependencies: `npm install`
   - Add API key to server.js
   - Run: `node server.js` or double-click `start.bat`

2. **Demo**:
   - Open http://localhost:3000
   - Click "Start Chat"
   - Ask about Revolt Motors
   - Interrupt AI by speaking or clicking "Interrupt"
